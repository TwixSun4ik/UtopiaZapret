Utopia Zapret — это легковесная графическая оболочка (GUI), написанная на Python, созданная для автоматизации запуска средств обхода блокировок (таких как zapret, goodbyedpi и др.) и мессенджера Discord в один клик.

⚠️ Дисклеймер (Важно!)
Данная программа НЕ ЯВЛЯЕТСЯ новым методом обхода или модификацией алгоритмов DPI.
Это всего лишь инструмент автоматизации.
Программа создана для того, чтобы избавить пользователя от необходимости вручную заходить в папки, запускать скрипты от имени администратора, а затем отдельно открывать Discord.
Программа работает только с файлами, находящимися в папке discord, обеспечивая базовую безопасность.

🚀 Как это работает
Вместо последовательности действий:
Открыть папку -> Найти .bat/.exe -> Запуск от админа
Найти ярлык Discord -> Запуск от админа
Вы просто нажимаете кнопку "АКТИВИРОВАТЬ UTOPIA".

🛠 Установка и запуск
Требования
Windows 10/11
Наличие любой альтернативы zapret-discord-youtube (или аналогичных скриптов).
Использование
Скачайте последнюю версию из Releases.
Запустите UtopiaZapret.exe.
Укажите путь к вашему .bat или .exe файлу внутри папки discord.
Нажмите кнопку запуска.

🛡 Безопасность
Программа имеет встроенный блок, запрещающий выполнение любых файлов вне папки discord, что предотвращает случайный запуск вредоносного ПО через интерфейс утилиты.
Разработано для личного пользования и удобства сообщества.
